What We Have Done So Far
We have collected a dataset covering U.S. mortgage lending from 2018 to 2021, merging tract-level HMDA data with county-level economic indicators from FRED. The raw data have been cleaned and consolidated so they are ready for visualization. We created the dashboard that visualizes mortgage approval patterns, minority borrower share, and related demographic metrics across counties. The dashboard has interactive mapping, filtering, and temporal navigation, allowing users to explore variations across space and time. We organized the project files carefully to ensure portability and static hosting compatibility. Generally, we have built a working prototype that demonstrates the core functionality we want to build our final project on.
What Will Be Most Challenging and Why
One of the biggest challenges in the next stage of our project will be building a machine learning style feature that provides users with dynamic, data-driven insights based on their inputs. Implementing this requires developing an algorithm that can take information such as county, demographic patterns, or loan characteristics and compare it to historical HMDA trends in a meaningful way. Because mortgage data is sensitive and heavily imbalanced across regions and groups, creating an ML model that is fair, interpretable, and statistically sound will be difficult. Another key challenge will be integrating this predictive or smart component into the existing dashboard without slowing down the interactive map or overloading the browser with large datasets. We also need to design a clear interface so users understand how the model is generating its outputs and what assumptions it relies on. Overall, the hardest part is making a feature that feels intelligent and helpful while still being transparent, accurate, and ethically responsible.
